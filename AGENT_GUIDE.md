# Agent Guide

## Purpose

Provide basic instructions for AI agents working on the BaoProject repository.\

## Scope

- Follow repository conventions.
- Use existing code style and patterns.
- Keep contributions minimal and safe.

## Basic workflow

1. Review issue or task.
2. Inspect relevant files.
3. Make targeted changes.
4. Validate with tests or linting if available.
5. Document any design decisions briefly.

## Communication

- Always read relevant guides in docs
- Be explicit about assumptions.
- Label uncertain areas for human review.
- Avoid changing unrelated files.

## Safety

- Do not introduce unsafe or insecure code.
- Do not include private data.
- Keep outputs concise and factual.
